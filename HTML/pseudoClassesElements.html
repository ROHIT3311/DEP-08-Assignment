CSS Pseudo-Classes and Pseudo-Elements Demo

:hover

Hover over the link to change its color:

Hover me!

The :hover pseudo-class is used to apply styles when an element is hovered over.

:focus

Click into the input field to change its background:

The :focus pseudo-class applies styles when an element (such as a text field) gains focus.

:active

Click on the link to apply an active style:

Click me!

The :active pseudo-class applies styles when an element is activated, like when clicked.

The :last-child pseudo-class targets the last child element of its parent.

The :nth-last-child(n) pseudo-class targets the element from the end, at the specified position.

:checked

Check the checkbox to change its background color:

The :checked pseudo-class targets input elements that are checked (like checkboxes or radio buttons).

:disabled

The disabled input will have a different background:

The :disabled pseudo-class targets input elements that are disabled.

:enabled

The enabled input will have a different background:

The :enabled pseudo-class targets input elements that are enabled.

:required

This input is required:

The :required pseudo-class targets input elements that are required.

:optional

This input is optional:

The :optional pseudo-class targets input elements that are optional.

:empty

This div has no content and will be hidden:

The :empty pseudo-class targets elements that have no children, including text nodes.

:root

The root element is targeted:

Root div

The :root pseudo-class targets the highest level parent element (usually html in HTML).

:target

::before

This paragraph will have content before it:

This is a paragraph.

The ::before pseudo-element inserts content before the content of the element.

::after

This paragraph will have content after it:

This is another paragraph.

The ::after pseudo-element inserts content after the content of the element.

::first-letter

This is the first letter of the paragraph:

This is a paragraph with a styled first letter.

The ::first-letter pseudo-element targets the first letter of the text in an element.

::first-line

This is the first line of the paragraph:

This is a paragraph with a styled first line.

The ::first-line pseudo-element targets the first line of the text in an element.

::selection

This text will change color when selected.

The ::selection pseudo-element applies styles to the selected text within an element.
