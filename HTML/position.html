Whenever you apply top, left, right and bottom css to an element then it will set according to its parent element. If we have div in that we have 4 boxes and box 2 will give position absolute and top and bottom. And we have not applied any position property to that div that is parent then that box will search above to grand parent and alas if not found, it will take html tag as parent and then apply the top and left So, if we want to apply such css according to parent then parent should have any position except static which is default

If that parent can have css like transform, filer and perspective properties can also make its child element appear as position by left and botton or any Means it will take position as parent not to its grand parent or html
